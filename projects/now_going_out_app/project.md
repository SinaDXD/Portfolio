# Now Going Out App

Source: https://sinadesign.ch/works/now-going-out-app
Year: 2020
Collaboration: Vala Design Studio
Services: UI/UX Design, Web Development, Digital Design
Duration: 8

## Short description
NOW is an interactive leisure-planning app.

## Overview
NOW is an interactive leisure-planning app that helps users discover events and places based on interests, budget, lifestyle, and available time. Users can plan alone or with friends. In group mode, the app suggests options that match everyone’s preferences, supports voting, and helps complete the plan through ticket purchase, booking, ride request, or navigation.

## Challenge
A core challenge was mobile-first usability, since most users were expected to access the service on their phones. The target audience included international tourists, so the project also needed accessible navigation across language barriers and compact information structure.

## Solution
The concept used mobile-first design and clear navigation structures. A hamburger menu gave shortcuts to page sections, and an AI-powered translation feature supported users switching between languages.

## Result
The final concept offers a smoother way to plan spontaneous outings with friends or people with similar tastes, moving quickly from “What should we do?” to a shared plan.
