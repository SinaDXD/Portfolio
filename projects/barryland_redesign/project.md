# Barryland Redesign

Source: https://sinadesign.ch/works/barryland-redesign
Year: 2026
Collaboration: ECAL
Services: UX Design, UI Design
Duration: 14 Weeks

## Short description
Redesign of the Barryland website developed in the Pixel Perfect course, focusing on improving engagement, usability, and visual expression through a more playful and family-oriented user experience.

## Overview
This project reinterprets the Barryland website based on prior UX and sitemap analysis. While the original site had a clear structure, it lacked interactivity and emotional engagement. The redesign introduces a warmer visual language, improved information flow, and interactive elements that encourage exploration while maintaining clarity.

## Challenge
Although Barryland’s existing website was clear and well-structured, it felt too rigid and serious for a brand centered around playful dogs and family experiences. The challenge was to make the digital experience feel more lively, engaging, and emotionally connected to the museum, while keeping the navigation clear and easy to use.

## Solution
To translate Barryland’s lively museum experience into a digital space, I designed a playful component system using Atomic Design methodology, inspired by the movement, energy, and joy of dogs. The visual language combines bold colors, dynamic layouts, motion cues, and frame-breaking elements. I also analyzed and refined the sitemap, simplified navigation, and designed a floating action button inspired by Barryland’s iconic barrel.

## Result
The redesign created a more immersive and intuitive website experience that better reflects Barryland’s identity. The improved sitemap made navigation clearer, while the design system brought consistency and emotional character across the platform.
